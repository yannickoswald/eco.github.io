Chapter 0 — Introduction to Python

Up-and-running with the essentials you’ll use throughout the book

Why Python?

Python is open-source, readable, and comes with a vast ecosystem of scientific libraries (NumPy, pandas, matplotlib, scikit-learn, xarray, …) that we will rely on for data wrangling, modelling and visualisation. If you are new to programming, Python’s gentle learning curve makes it an ideal first language.

Getting Set Up

  • 1. Install Python
  • 2. Choose an Editor / IDE
  • 3. Run Your First Script

1 · Install Python (≥ 3.12)

The simplest route is to install the Anaconda distribution, which bundles Python itself plus common data-science packages. Alternatively, download the official installer from python.org and manage packages with pip or pipx.

2 · Editors and Interactive Notebooks

You can write Python in any text editor, but modern IDEs such as VS Code, PyCharm Community, or browser-based Jupyter Lab (shipped with Anaconda) give you syntax highlighting, auto-completion and an interactive console — all of which speed up learning.

3 · Hello, World!

Save the snippet below as hello.py and run it from a terminal:

print("Hello, ecological economists!")

Then execute:

$ python hello.py
Hello, ecological economists!

Core Language Concepts

Variables and Basic Types

year       = 2025            # int
growth     = 3.2             # float
descriptor = "biophysical"   # str
renewables = True            # bool

Lists, Tuples, Dictionaries

countries   = ["CH", "DE", "FR"]                     # list
coords      = (47.38, 8.54)                          # tuple
gdp_musd    = {"CH": 907000, "DE": 4470000}          # dict

Control Flow

for c in countries:
    if c in gdp_musd:
        print(c, gdp_musd[c])

Functions

def petajoules_to_twh(pj):
    """Convert energy from PJ to TWh"""
    return pj * 0.277778

print(petajoules_to_twh(1500))  # 416.7

Working with Packages

Use pip install numpy pandas matplotlib (or conda install …) and then: 

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

We will introduce each of these libraries as we progress, but for now make sure you can import them without errors.
